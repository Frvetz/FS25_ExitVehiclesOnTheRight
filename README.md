# FS25_ExitVehiclesOnTheRight

## English description

With this mod, you can now exit your vehicles on both sides. Very useful if you are stuck against a wall and cannot get out! Just get out on the other side!

- Default key combination: L-Shift + E

- Multiplayer compatible


## Deutsche Beschreibung

Mit diesem Mod können Sie Ihre Fahrzeuge nun auf beiden Seiten verlassen. Sehr nützlich, falls sie an einer Wand stehen und nicht aussteigen können! Einfach auf der anderen Seite aussteigen!

- Standardmäßige Tastenkombination: L-Shift + E

- Multiplayer kompatibel
