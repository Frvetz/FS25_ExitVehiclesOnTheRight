# FS25_ExitVehicleOnTheRight

## English description

With this mod, you can now exit your vehicle on both sides.

-Default key combination: L-Shift + E

-Multiplayer compatible

Mit diesem Mod können Sie Ihr Fahrzeug nun auf beiden Seiten verlassen.

-Standardmäßige Tastenkombination: L-Shift + E

-Multiplayer kompatibel
